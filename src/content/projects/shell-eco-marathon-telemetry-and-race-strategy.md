This project integrates live monitoring and predictive strategy for Swarthmore's hydrogen fuel-cell Shell Eco-marathon prototype vehicle.

## What it includes

- Flask and Socket.IO dashboard for real-time telemetry visualization
- RS232/serial packet ingestion and decoded fuel-cell monitoring
- JSON Lines logging for test-session replay and post-run analysis
- Track-aware burn-and-coast strategy engine for speed, timing, and energy estimates
- Static web simulator for strategy exploration
