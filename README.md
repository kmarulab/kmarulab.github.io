# kmarulab.github.io

Personal and academic portfolio for Emmanuel Kimaru Boruett.

- **Theme:** [Academic Portfolio Astro](https://github.com/rubzip/academic-portfolio-astro)
- **Framework:** [Astro](https://astro.build/) + [Tailwind CSS](https://tailwindcss.com/)

## Running Locally

This project requires **Node.js >= 22.12.0**. To preview the site locally before pushing to GitHub Pages:

### 1. Install Dependencies
```bash
npm install

### 2. Start Development Server
```bash
npm run dev
```
The site will be available at http://localhost:4321.

### 3. Build for Production
```Bash
npm run build
```
